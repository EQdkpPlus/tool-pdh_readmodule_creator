This pdh read-module generator is for EQdkp Plus 2.0.

USAGE:
=============
Put this file into the root director of your  EQdkp Plus Installation.
Open the file and make your settings.
Save the file and open it in your browser, e.g. http://localhost/eqdkp/pdh_readmodule_creator.php
You will find the output file in the same folder of this script.
